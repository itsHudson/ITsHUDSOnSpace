---

# 1.1.1 What is SQL

## What is SQL?

SQL stands for:

**Structured Query Language**

It is a language used to communicate with relational databases.

A database stores information in tables.

Example:

| student_id | name | age |
|------------|--------|------|
| 1 | John | 20 |
| 2 | Sarah | 21 |

SQL allows us to:

- retrieve data
- insert data
- update data
- delete data
- create tables
- manage permissions
- control transactions

---

## Why SQL was created

Before SQL:

- data was harder to manage
- businesses used manual record systems
- retrieving information was slow

SQL created a standardized way to manage structured data.

---

## How SQL works

Step 1 → User writes SQL query  
Step 2 → Database receives query  
Step 3 → SQL parser checks syntax  
Step 4 → Optimizer creates execution plan  
Step 5 → Database retrieves/modifies data  
Step 6 → Results are returned

---

## Basic Example

```sql
SELECT name
FROM students;
```

Explanation:

- `SELECT` → choose data
- `name` → specific column
- `FROM students` → target table

Output:

| name |
|--------|
| John |
| Sarah |

---

## What SQL can do

| Operation | Purpose | Example |
|------------|----------|----------|
| Retrieve | Get data | `SELECT` |
| Insert | Add data | `INSERT` |
| Update | Modify data | `UPDATE` |
| Delete | Remove data | `DELETE` |
| Create | Build structures | `CREATE TABLE` |
| Security | Manage permissions | `GRANT` |
| Transactions | Manage changes | `COMMIT` |

# 1.1.4 SQL vs Programming Languages

Many beginners confuse this.

SQL is NOT a full programming language like Python.

---

## SQL

SQL is declarative.

You tell database:

**what you want**

Example:

```sql
SELECT name
FROM employees;
```

---

## Programming Languages

Examples:

- Python
- Java
- JavaScript
- C++

These languages tell computer:

**how to do something**

Example:

```python
for employee in employees:
    print(employee.name)
```

---

## Comparison Table

| Feature | SQL | Programming Languages |
|----------|------|------------------------|
| Type | Declarative | Procedural/OOP |
| Focus | Data | Applications |
| Usage | Databases | Software development |
| Example | SQL | Python |

---

## Real-world Example

# 1.1.5 SQL vs NoSQL

This comparison is very important.

---

## SQL Databases

Examples:

- MySQL

- Oracle

---

### Strengths

- strong consistency
- ACID transactions

- structured schema

---

## NoSQL Databases

Examples:

- MongoDB
- Redis
- Cassandra

---

|----------|------|--------|
