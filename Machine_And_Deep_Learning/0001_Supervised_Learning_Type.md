# 1. DEFINITION

| Concept | Definition | How It Works | Example |
| --- | --- | --- | --- |
| Supervised Learning | A type of machine learning where a model learns from labeled input-output data to predict future outputs. | The model analyzes many examples of X and Y to learn patterns that map inputs to outputs, then applies those patterns to new data. | Email → Spam / Not Spam |
| Input (X) | The data provided to the model for making predictions. | The model receives input variables and processes them to generate an output prediction. Inputs may be numerical, text, images, audio, etc. | House size = 1000 sqft |
| Feature | Individual measurable attributes inside the input data. | Each feature provides useful information that helps the model detect relationships with the output. Multiple features form one input record. | House size, location, number of rooms |
| Output (Y) | The correct answer associated with each input during training. | The model compares predicted output with actual output to measure error during learning. | House price = RM300,000 |
| Label / Target | The expected output value the model is trying to predict. Same meaning as output Y. | Labels act as the reference answer that guides the model during training. | Spam / Not Spam |
| Dataset | A collection of multiple input-output records used for machine learning tasks. | The dataset provides examples for training, validation, and testing the model. | 10,000 customer records |
| Labeled Data | Data where each input has a known correct output label. | The model learns relationships by studying both features and their corresponding labels. | Email text + spam label |
| Mapping Function f(X)=Y | Mathematical relationship that converts input into predicted output. | The model gradually learns this function by adjusting internal parameters during training. | f(size, location) = house price |
| Prediction | The output generated by a trained model for new unseen input data. | After training, the model applies learned patterns to estimate outputs for data it has never seen before. | Predict price of a new house |
